## CORNET (Flagship) — `rbccps-iisc/CORNET`

- **Stack**: Generic plugin-based framework; NS-3 5G NR + Mininet-WiFi backends; Gazebo Classic 11 + ROS 2 Humble robot backend
- **Python package**: `pip install cornet-framework`
- **URL**: <https://github.com/rbccps-iisc/CORNET>
- **Status**: Active development.

Plugin code is ported (copied and adapted) from CORNET 2.0 and 3.0. Neither predecessor repo was modified.
